Below is a developer-ready, broken-down feature list that combines what you see across Bigo Live / Tango / Superlive / Clapper-style apps, organized as FEATURES FOR IGNYTLOVE so Replit can build it cleanly.

0) Product Scope for Replit MVP

Mobile-first app with:

Auth + profiles

Browse live users

Go live (host)

Public chat + likes

Gifts (coins/tokens)

1:1 private messaging

Private video chat billing (per-minute AND per-session)

Wallet + payments

Moderation + reporting

Notifications

1) User Roles & Permissions
Roles

Guest (not logged in)

Browse limited public content (optional)

User (viewer)

Watch lives, chat, gift, follow, DM, request private call

Creator/Host

Go live, receive gifts/earnings, manage room mods

Moderator (room-level)

Mute/ban users in a room, delete messages, pin messages

Admin (platform-level)

Full moderation, payouts, disputes, content enforcement

Core Permission Rules

Only verified/eligible users can host (configurable).

Users can be restricted (chat disabled, live disabled, payment disabled).

2) Authentication & Account
Features

Register: phone/email + password OR OTP (choose one for MVP)

Login / logout

Forgot password / OTP login

Device/session management (basic)

Account deletion request (basic)

Basic age gate (18+ recommended for live platforms)

Data Fields

user_id, phone/email, username (unique), password_hash/otp, created_at, status

3) Profiles (Viewer + Creator)
Viewer Profile

Avatar, display name, username

Bio, location (optional), interests/tags

Followers/following counts

Level/badges (optional for MVP)

Wallet balance (coins)

Settings: privacy, notifications, blocked users

Creator Profile (Host)

All viewer fields +:

Host level / stats: total lives, total gifts, earnings

Payout method info (admin-only visibility)

“Available for private call” toggle

4) Discovery & Home Feed (Browse Live People)
Core Screens

Home feed: live rooms + recommended creators

Tabs: Following | For You | Nearby (optional) | Categories

Search: creators, rooms, keywords, tags

Live room cards: host avatar, title, viewers, country, “LIVE”

Sorting/Ranking (simple for MVP)

By viewer count, trending score, followed creators first

Filters

Language, country, category, “private call available”

5) Live Streaming (Public)
Host: Go Live Flow

Create live room:

Title, category, language, audience (public)

Optional: tags, location

Start stream (RTC streaming)

End stream

Host controls:

Mute user, ban user, add moderator

Pin message, enable slow mode (optional)

Toggle chat on/off (optional)

Toggle gifts on/off (optional)

Viewer: Watch Flow

Join live room

View stream + host info

Send chat messages

Send likes/hearts

Send gifts

Follow host

Share room link

Room Requirements

Real-time viewer count

Join/leave events

Basic reconnect handling

6) Public Chat + Reactions
Public Chat (in live room)

Text messages

Emoji

Message rate limit (anti-spam)

Delete message (mods/admin)

Reactions

Like/heart spam-tap with throttling

On-screen animations (simple MVP)

System Messages

“User joined”

“User followed host”

“Gift sent”

7) Gifts (Coins/Tokens Economy)
Gift Catalog

Gift list with: gift_id, name, coin_cost, animation_key

Basic tiers: small/medium/large

Limited gifts (optional)

Sending Gifts

Viewer selects gift → confirms → coins deducted → gift event broadcast in room

Host receives gift value as earnings (separate ledger from coins)

Wallet Ledger (must-have)

Every transaction stored:

user_id, type (purchase/spend/earn/refund), amount, reference_id, timestamp

8) Payments & Wallet Top-Up
MVP Options

In-app purchases (mobile) OR Stripe (web-like) depending on platform choice

Purchase coin packs

Core Requirements

Wallet balance view

Purchase history

Failed payment handling

Anti-fraud flags (basic)

9) 1:1 Private Messaging (DM)
Features

DM inbox list (recent chats)

1:1 text messages

Message requests (optional)

Block user

Report conversation/user

Nice-to-have (later)

Images, voice notes, read receipts

10) Private Video Chat (Paid) — PER-MINUTE + PER-SESSION

This is a key “Tango/Bigo-style” monetization feature.

Private Call Types

Per-minute billing

Rate: coins/minute (creator sets, or platform sets)

Billing starts when call connects

Billing ticks every minute (or every N seconds prorated)

Auto-end call when balance insufficient (with warning)

Per-session billing

Flat fee per session (e.g., 500 coins for 10 minutes OR per session without fixed time)

Optional “session duration limit”

Request → Accept Flow

Viewer taps “Private Call”

Chooses billing mode (minute/session)

Shows price + available balance

Sends request to host

Host can Accept/Decline

If accepted:

Coins pre-authorized (hold) or immediate charge

Start RTC 1:1 video room

During Call

Countdown / timer visible

Warning when balance low

End call button for both

Auto-end conditions:

viewer ended

host ended

network timeout

insufficient funds

Records

call_id, viewer_id, host_id, start_time, end_time, billing_mode, rate, total_charged, status

11) Following, Fans, and Social Layer
Features

Follow/unfollow creators

Followers list / following list

“Following feed” shows when creators go live

Basic badges/levels (optional for later)

12) Notifications
Push/In-app notifications

Creator goes live (followed)

DM received

Private call request received

Purchase success/fail

Account warnings/mod actions

13) Moderation, Trust & Safety (Must-have)
In-room moderation tools

Mute user (time-based)

Kick user

Ban user (room or platform)

Delete messages

Slow mode / chat rate limit

Reporting

Report user

Report live room

Report message

Report private call abuse

Blocking

Block user (prevents DM, prevents joining private call, optional hides rooms)

Admin Panel (basic)

User management: suspend/ban/unban

Review reports queue

View transaction ledger

Creator payouts management (later)

14) Content Policies & Safety Controls (Practical MVP)

Age gate toggle

Prohibited content list (config)

Auto-moderation placeholders (later)

Stream delay (later)

Screenshot/recording policy notice (legal)

15) Analytics (Minimal but useful)
Creator metrics

Total viewers, peak viewers, gifts earned, call revenue

Platform metrics

Daily active users, total revenue, top creators, top rooms
